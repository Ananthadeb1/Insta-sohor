Difference between Local-Storage and Session-Storage

Whenever a tab is opened in a browswer it require some storage. Storage can be of two types-
1)Local-Storage
2)Session-Storage
Session-Storage is a type of storage that keeps the information during the session. Means if you go backwards close the tab it will clear the informations.A page session lasts as long as the tab or the browser is open, and survives over page reloads and restores.
But Local-Storage is a type off storage that keeps the information even if you go backwards or close the tab except you close all the tabs which means closing the browser.Local-Storage is similar to Session-Storage, except that while localStorage data has no expiration time.

What is Event Loop ???

Event loop is a process of compiling asynchronus code in browser. In javascript codes ar compiled one by one. But in case of setTimeout() there is a time declared after which that code gets executed. In browser every line of code goes to the stack one by one and gets executed one after another.If any code like setTimeout() or fetching API comes in execution and it causes delay then browser puts it aside and goes for the next code and next code till that delaying one is ready to execute. The time its ready to execute and if stack is empty that time this code goes to stack and gets execute. Thus asynchronus codes get queued and gets executed later as a loop and its simply called event loop.
